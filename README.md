## 目录结构

```
-------------------------------------------------------|
src
|-+api
|   |--proxyFetch.js          // fetch的封装
|   |
|-+components                 // 组件按照页面文件结构来
|   +--common
|   |   |...
|-+assets                     // 静态资源， 图片或者字体在build会被直接拷到dist目录下
|   +--images
|   |   |...
|-+containers
|   +--home                   // 首页模块
|   |   |...
+--utils                      // 封装得插件或者工具类
|   +--
+--stores                     // redux相关
|   |
|-+styles
|   |--base.scss
```
